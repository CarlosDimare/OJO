# Assets

Los siguientes archivos son descargados automáticamente por Dockerfile.build:

- alpine-arm64.tar.gz       → Alpine Linux 3.19 ARM64 rootfs
- bun-linux-aarch64.zip     → Bun runtime para ARM64
- proot-static-arm64        → proot binario estático ARM64

No es necesario ponerlos manualmente si usás BUILD.sh
